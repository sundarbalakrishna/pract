# Set the center of our map ("m") with the first pair of coordinates and set the projection to cylindrical;
# For documentation, see https://jakevdp.github.io/PythonDataScienceHandbook/04.13-geographic-data-with-basemap.html 
m = Basemap(lat_0 = lats[0], 
            lon_0 = lngs[0], 
            projection ='cyl') 

# The first data point is chosen to be the centre of the basemap, is my arbitrary decision;
